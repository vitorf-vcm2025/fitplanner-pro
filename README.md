## 🚀 Sobre o Projeto

O **FitPlanner PRO** é um software de desktop moderno desenvolvido em **Python** com uma interface gráfica premium em **Dark Mode**. Ele foi projetado especificamente para eliminar o uso de cadernos e planilhas lentas, permitindo que profissionais da área fitness escalem seus negócios e entreguem uma experiência exclusiva para seus alunos.

O sistema conta com armazenamento local estruturado e dinâmico, garantindo rapidez e segurança no gerenciamento das informações.

---

## ✨ Funcionalidades Atuais (Versão 1.0)

- **📊 Dashboard Inteligente:** Painel principal com visão geral dos alunos ativos e atalhos rápidos de navegação.
- **💪 Gestão Dinâmica de Treinos:** Criação e listagem de treinos customizados com foco, objetivo e duração.
- **💾 Banco de Dados Integrado (JSON):** Cadastro e persistência de novos planos de treino em tempo real diretamente no arquivo local `data/treinos.json`.
- **🍏 Módulo de Nutrição (Dieta):** Visualização estruturada de planos alimentares, controle macro (calorias, proteínas) e horários de refeição.
- **📈 Acompanhamento de Progresso:** Histórico visual de avaliações físicas contendo evolução de peso, percentual de gordura corporal (%GC) e massa magra.

## 🚧 Próximamente (Versão 2.0) - _Roadmap_

- **📅 Agenda Integrada:** Sincronização automatizada de horários de atendimento dos alunos.
- **⚙️ Automações Premium:** Envio de alertas e notificações de treinos diretamente para o WhatsApp do aluno.

## 🛠️ Tecnologias Utilizadas

- **Language:** [Python 3](https://www.python.org/)
- **GUI Framework:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) (Interface moderna e responsiva)
- **Data Storage:** JSON (JavaScript Object Notation) para persistência local leve.
